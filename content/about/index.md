## 恒健知行 - 个人博客

本站是微信公众号「恒健知行」内容的独立镜像站点。

### 技术栈

- **Hugo** - 静态站点生成器
- **GitHub Pages** - 免费托管
- **GitHub Actions** - 自动构建部署

### 部署

每次推送到 `main` 分支后，GitHub Actions 会自动构建并发布到 GitHub Pages。
